# Team Responsibilities

This folder tracks who owns what during the 3-day Corvus MFG lean rebuild sprint.

The sprint has one shared product workflow:

```text
manufacturing data -> risk detection -> daily brief -> owner questions
```

## Sprint Rule

If a task does not support ingestion, risk analysis, brief generation, demo output, or customer validation, it should wait until after the MVP.
